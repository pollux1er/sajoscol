GEPI - Typographical short cuts

  • Make lists or enumerations

    To make lists, return to the line and to begin the new line with a star (« * »).
    It is possible to make numbered lists by using # in the place of star :

  • Bolde, italic and underlined

    Indicates simply text in italic by placing it between the symbols '' (i.e., twice the symbol "apostrophizes"). Thus, if you type: « ''Here text in italic'' », you will obtain : « Here text in italic ».

    Underlines text while placing it between the symbols ''' (i.e., three times the symbol "apostrophize"). Thus, if you type: " ' ' ' Here underlined text ' ' ' ", you will obtain: « Here underlined text ».

    Indicates text in fat while placing it between the symbols __ (twice character of underlining).

  • Centered text

    It is possible to center text while placing it between the symbols _ C_.

  • Put titles of various levels

    Makes precede the titles by the symbols !, !!, or !!!, according to the size of the title wished.

  • Put text color

    Indicates for example the text to be put in red while placing it between the symbols _r_ . Thus, if you type : « _r_Here red_r_ », you will obtain : « Here red ».
    Other recognized colors :
    _b_ for blue
    _v_ for the green
    _o_ for orange
    _m_ for chestnut
    _j_ for the yellow
    _p_ for the crimson
    _g_ for the gray

  • The hyperlinks

    The hyperlinks are directly recognized by GEPI. Thus, if you type : « http://www.google.fr. », you will obtain « http://www.google.fr. ».
    Moreover, a personalized hyperlink can be easely make with the following code : « [The search engine google|http://www.google.fr] » becomes « The search engine google ».
    In the same way it is possible to insert an image by typing a code such as this one : [http://www.monsite.fr/image.jpg].

  • Return to the line

    A simple return to the line in the code does not cause a return to the line on the screen.
    To obtain a return to the line :
    - or it is enough to add at the end of the line the symbols %%% (three times the symbol "percentage"),
    - or leaves a blank line. In this case, the return to the line will be accompanied by a jump of line.

  • To insert HTML tags

    It is possible to insert HTML tags in the text, but so that it are interpreted correctly, it is necessary to make begin each line with the symbol « | »
