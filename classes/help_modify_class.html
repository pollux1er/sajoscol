GEPI - Management of the classes: help on the courses

Sur this page, you define the courses taught in the class and the professors who teach them.
1/ Begins by check the taught courses.
2/ Once that one validated, one specifies the professors who teach the course in the class. It is possible to assign an unspecified number of professorrs for a course. At each validation, a new field of typing appears allowing the assignment of a new professor. CAUTION : if no professor is affected, the course is not recorded !
3/ then you can (de)select the students concerned by the discipline. By defect all the students follow the discipline, you can click on the link "Registered students" to make appears a list allowing to (de)select certain students.
  • Registered Students

    This link opens a window making it possible to visualize/modify the options of each student, for the course concerned.
    The number into brackets display beside the mention "registered Students" indicates the number of students registered in the courses for thefirst period.

  • Priority of display

    Allows to define the order of display of the courses in the report card and the summary tables of the averages.
    Remarks :
    • It is possible to allot the same weight to several courses not appearing on the same bulletin. For example, all the LV1 can have the same weight, etc.
    • If two courses appearing on the same bulletin have the same priority, GEPI display the first course extracted from the base.


  • Coefficient

    Allows to calculate for the general average of the class and the general average of each student. These values are displayed in the report card.

    1/ If all the coefficients for a class are equal to zero, there is no calculation of the general average and, consequently, not of display of an additional line on the bulletin.

    2/ if at least a coefficient is nonnull, an additional line is displayed on the bulletin, indicating the averages of the student and class on all the affected courses of a coefficient not null.
